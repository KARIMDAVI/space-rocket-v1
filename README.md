## Game Status

- Status: Playable and active
- Engine: Three.js (WebGL)
- Input: Mouse + touch support
- Objective: Dodge incoming asteroids and survive
- Progression: Difficulty increases over time (faster movement + more obstacles)

## Game Peaks (Highlights)

- Real-time scoring system (`SCORE`)
- Best score tracking (`BEST`)
- High score is stored locally in browser `localStorage`
- Crash/restart loop with instant retry
- Responsive HUD with live status messages

## How to Play

1. Open `space-rocket.html` in your browser.
2. Click/tap and hold to boost.
3. Move to dodge asteroids.
4. If you crash, click/tap to restart.

## Controls

- Mouse move / touch move: steer rocket target
- Mouse down / touch start: boost
- Mouse up / touch end: release boost

## Project Files

- `space-rocket.html` – Main HTML shell and HUD
- `space-rocket.css` – UI and visual styles
- `space-rocket.js` – Game logic, rendering, collisions, scoring
