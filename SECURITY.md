# Security Policy for Space Rocket V1

## Supported Versions
| Version | Supported          |
|---------|--------------------|
| 1.0.0   | :white_check_mark: | 
| 1.1.0   | :white_check_mark: | 
| 1.2.0   | :x:                |

## Reporting Vulnerabilities

If you discover a vulnerability in this project, please report it as soon as possible. 

### Preferred Contact Method
You can reach the security team via email at security@karimdavi.dev.

### Expected Response Times
We will acknowledge your report within 72 hours and aim to provide a detailed response within 7 business days.

### Disclosure Policy
We ask that you do not disclose your findings publicly until we have had a chance to address them. During this time, we will work to resolve the issue and keep you updated.

### Requesting a CVE
If a vulnerability is confirmed, you may request a CVE (Common Vulnerabilities and Exposures) ID through the appropriate channels. Please include all relevant information in your report so that we can process your request efficiently.
